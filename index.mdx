StageSnap provides a REST API for generating AI-powered real estate images and videos.
Use it to stage rooms, produce marketing flyers, and create listing videos—all
programmatically, at scale.

## Base URL

All API requests go to:

```
https://stagesnap.ai/api/v1
```

Use the staging environment for development and testing:

```
https://staging.stagesnap.ai/api/v1
```

## Authentication

All endpoints require a bearer API key. See [Authentication](/authentication) for setup.

## Response format

All responses are JSON. Successful responses include `"success": true`. Errors
include `"success": false` and an `"error"` string.

```json
// Success
{ "success": true, "image": "<base64>", "model": "gemini-2.5-flash-preview-image" }

// Error
{ "success": false, "error": "Either imageUrl or imageBase64 is required" }
```
