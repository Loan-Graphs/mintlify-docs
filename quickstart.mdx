This guide walks you through staging a room photo using the StageSnap API.

## Prerequisites

- A StageSnap account at [stagesnap.ai](https://stagesnap.ai)
- An API key (see [Authentication](/authentication))

## Stage a room

<Steps>
  <Step title="Get a source image">
    You need a URL or base64-encoded JPEG/PNG of the room you want to stage.
    For this example we use a public URL.
  </Step>

  <Step title="Send a staging request">
    Call `POST /api/v1/stage` with your image and desired style.

    <CodeGroup>
      ```bash cURL
      curl -X POST https://stagesnap.ai/api/v1/stage \
        -H "Authorization: Bearer sk_live_your_key_here" \
        -H "Content-Type: application/json" \
        -d '{
          "imageUrl": "https://example.com/empty-living-room.jpg",
          "style": "modern",
          "roomType": "living_room"
        }'
      ```

      ```javascript Node.js
      const response = await fetch('https://stagesnap.ai/api/v1/stage', {
        method: 'POST',
        headers: {
          'Authorization': 'Bearer sk_live_your_key_here',
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({
          imageUrl: 'https://example.com/empty-living-room.jpg',
          style: 'modern',
          roomType: 'living_room',
        }),
      });

      const data = await response.json();
      console.log(data.image); // base64 JPEG
      ```

      ```python Python
      import requests, base64

      response = requests.post(
          "https://stagesnap.ai/api/v1/stage",
          headers={"Authorization": "Bearer sk_live_your_key_here"},
          json={
              "imageUrl": "https://example.com/empty-living-room.jpg",
              "style": "modern",
              "roomType": "living_room",
          },
      )

      data = response.json()
      # Save the result
      with open("staged.jpg", "wb") as f:
          f.write(base64.b64decode(data["image"]))
      ```
    </CodeGroup>
  </Step>

  <Step title="Use the result">
    The response includes a base64-encoded JPEG in the `image` field. Decode and
    display or save it.

    ```json
    {
      "success": true,
      "taskId": "task_abc123",
      "image": "/9j/4AAQSkZJRgAB...",
      "model": "gemini-2.5-flash-preview-image"
    }
    ```
  </Step>
</Steps>
